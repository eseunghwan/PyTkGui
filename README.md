# Install
## using pip(stable)
```powershell
pip3 install PyTkGui
```
## using git(dev)
```powershell
pip3 install https://github.com/eseunghwan/PyTkGui.git
```
<br><br>

# Usage
### create project with cli
```powershell
pytkgui-cli init [dest]
```
<br>

### run ui
```powershell
pytkgui-cli run dev
```
<br>

### build project to dest
```powershell
pytkgui-cli run build [dest]
```
